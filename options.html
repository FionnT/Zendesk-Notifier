Adding Views:

1. Copy the URL of the filter you would like to add:

2. Go to the settings page, and input the URL in the filter field
3. Add a queue name for easy recognition/management later on!
4. Select the parameters for when you want to be notified (these are the same as Zendesk's custom filters)
5. Save the queue!
